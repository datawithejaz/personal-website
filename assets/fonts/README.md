# Self-hosted Neue Montreal Fonts

Download from: https://www.fontshare.com/fonts/neue-montreal

Place the following .woff2 files in this directory:

- `NeueMontreal-Regular.woff2` (weight 400)
- `NeueMontreal-Medium.woff2` (weight 500)
- `NeueMontreal-Bold.woff2` (weight 700)

After placing the files, the @font-face rules in main.css will load them locally
instead of from the Fontshare CDN.
